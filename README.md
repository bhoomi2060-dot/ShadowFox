## Task 1 — Beginner Level (Python Fundamentals)

**File:** `Task1 (Beginner Level).py`

A set of beginner Python exercises covering core language concepts, completed as part of the "Do 5 out of 9" beginner task sheet. Topics covered include:

- **Variables** — creating variables, checking data types, and understanding reserved keyword behavior (e.g. using `for` as a variable name); calculating Simple Interest.
- **Numbers** — using the `format()` function on numeric/string arguments; calculating circle area and derived values (e.g. pond water volume); computing speed from distance and time.
- **Lists** — manipulating a `justice_league` list: counting members, inserting/appending new members, reordering, replacing the list contents, and sorting alphabetically.
- **If Conditions** — BMI category calculator based on user input (height & weight); determining which country a given city belongs to; checking whether two cities belong to the same country.
- **Dictionaries** — building a list of friend-name tuples with name lengths; tracking trip expenses across two dictionaries (self vs. partner), calculating totals, comparing overall spend, and identifying the category with the largest spending difference.

**How to run:**
```bash
python "Task1 (Beginner Level).py"
```

## Task 2 — Web Scraper

**File:** `Task2.py`

A web scraper built with **Requests** and **BeautifulSoup** that extracts product data from the [Web Scraper e-commerce test site](https://webscraper.io/test-sites/e-commerce/allinone/computers/laptops).

**What it does:**
- Sends an HTTP request to the laptops category page.
- Parses the HTML and locates each product card (`div.product-wrapper`).
- Extracts for every product:
  - **Title**
  - **Price**
  - **Star Rating**
  - **Number of Reviews**
- Prints the scraped data to the console.
- Saves all results to `laptops.csv` for further use in Excel/Google Sheets or data analysis.

**Libraries used:**
- `requests` — for fetching page HTML
- `beautifulsoup4` — for parsing HTML and extracting data via CSS selectors
- `csv` — for exporting structured data

**How to run:**
```bash
python Task2.py
```

**Output:** Console printout of each scraped product + a generated `laptops.csv` file containing all results (117 products at time of writing).

## Task 3 — Hangman Game

**File:** `Task3.py`

A console-based implementation of the classic word-guessing game **Hangman**, featuring visual progress and a hint system.

**Features:**
- **Word bank with hints** — each word has an associated clue.
- **Visual progress** — ASCII-art gallows drawing that updates with each wrong guess (7 stages, from empty gallows to fully drawn figure).
- **One-time hint per game** — players can request a clue, but only once, adding a strategic tradeoff.
- **Input validation** — handles invalid/duplicate letter guesses gracefully.
- **Win/lose detection** — game ends when the word is fully guessed (win) or the maximum wrong guesses is reached (loss).
- **Replay loop** — prompts the player to play again after each round.

**How to run:**
```bash
python Task3.py
```

**Example gameplay:**
```
Word: _ _ _ _ _ _ _ _
Wrong guesses remaining: 6
Enter a letter (or 'hint'): e
Good guess! 'e' is in the word.
```
